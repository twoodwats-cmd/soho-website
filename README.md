# SoHi Grilled Sandwiches — Website

Single-page scroll-animation site for SoHi Grilled Sandwiches (Uptown Oxford, OH).
Static: `index.html` + `images/`. No build step. Deploys as-is on Vercel/Netlify.

- Hero: scroll-scrubbed explode-and-drop burger (GSAP + ScrollTrigger via CDN).
- Marquee: real photos, orange-framed, captioned.
- Full menu with prices; Order buttons → online ordering.
